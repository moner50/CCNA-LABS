## 📖 About This Repository

This repository documents my hands-on journey through CCNA topics. Each lab includes:
- 🗺️ **Network topology diagram**
- ⚙️ **Step-by-step configurations**
- 💡 **Key concepts and explanations**
- ✅ **Verification commands and expected output**

Whether you're studying for the CCNA exam or just want to sharpen your networking skills, feel free to explore and learn!

## 🛠️ Tools & Requirements

To open and run these labs you'll need one or more of the following:

- **[Cisco Packet Tracer](https://www.netacad.com/courses/packet-tracer)** — Free with a Cisco NetAcad account
- **[GNS3](https://www.gns3.com/)** — Free & open-source network simulator
- **[EVE-NG](https://www.eve-ng.net/)** *(optional)* — For more advanced topologies

## 🚀 How to Use

1. **Clone the repository**
   ```bash
   git clone https://github.com/YOUR-USERNAME/CCNA-LABS.git
   ```

2. **Navigate to any lab folder**
   ```bash
   cd CCNA-LABS/01-Basic-Configuration
   ```

3. **Read the `lab-guide.md`** for objectives and instructions

4. **Open the topology file** in Packet Tracer or GNS3

5. **Apply the configs** and verify using the provided commands
