# 🔁 OSPF Labs — Section 05

This section covers **Open Shortest Path First (OSPF)** labs from the CCNA curriculum.
OSPF is a link-state routing protocol that uses Dijkstra's algorithm to find the best path
and is one of the most important topics in the CCNA exam.

## 🧠 Key OSPF Concepts Covered

- OSPF neighbor relationships and adjacency
- Single-area OSPFv2 (Area 0)
- Point-to-point vs. multi-access network types
- Designated Router (DR) and Backup Designated Router (BDR) election
- OSPF Hello and Dead timer configuration
- OSPF cost metric, bandwidth adjustment, and route selection
- Verification commands: `show ip ospf neighbor`, `show ip route ospf`, `show ip ospf interface`
